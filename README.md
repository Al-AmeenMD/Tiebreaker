# Tiebreaker ⚖️

**Tiebreaker** is an AI-powered decision strategist designed to turn uncertainty into clarity. Whether you're choosing a career path, a new home, or a business strategy, Tiebreaker dissects the variables to help you make the most informed choice possible.

## ✨ Features

- **AI-Driven Insights**: Leverages Gemini 3 Flash to provide objective, well-reasoned analysis.
- **Pros & Cons**: Detailed breakdown of advantages and disadvantages for every option.
- **Comparison Tables**: Side-by-side evaluation of multiple choices across key metrics.
- **SWOT Analysis**: Strategic mapping of Strengths, Weaknesses, Opportunities, and Threats.
- **The Verdict**: A final recommendation and summary to guide your decision-making process.
- **Premium UI**: A refined, editorial-style interface built with React, Tailwind CSS, and Motion.

## 🛠️ Tech Stack

- **Frontend**: React 19, TypeScript, Vite
- **Styling**: Tailwind CSS 4
- **Animations**: Motion (formerly Framer Motion)
- **AI**: Google Gemini API (@google/genai)
- **Icons**: Lucide React

## 🚀 Getting Started

1. **Clone the repository**
2. **Install dependencies**:
   ```bash
   npm install
   ```
3. **Set up environment variables**:
   Create a `.env` file and add your Gemini API Key:
   ```env
   GEMINI_API_KEY=your_api_key_here
   ```
4. **Run the development server**:
   ```bash
   npm run dev
   ```
